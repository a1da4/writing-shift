# writing-shift

## Setting
1. Clone this repository
```
% git clone https://github.com/a1da4/writing-shift.git
```

2. Install Packages

 - Docker  

```
% docker build -t "writing-shift" .

% docker run -it "writing-shift"
```

 - PIP (use `requirements.txt`)  

```
% pip install -r requirements.txt
```
